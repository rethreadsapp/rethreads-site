01

服務說明

Rethreads（以下簡稱「本服務」、「我們」）提供一個網頁應用程式，讓使用者透過 Meta Threads 官方 API 授權，將自己帳號的貼文與回覆匯出、整理、備份，並可選擇發布至 Google Blogger 或以 PDF、Markdown 格式下載。

本服務目前提供免費方案及付費進階方案。各方案的功能範圍詳見第 4 條。

重要： 本服務僅能存取您主動授權的 Threads 帳號資料，且僅限您自己的帳號。本服務無法也不會存取他人的私人資料。
02

帳號與授權

使用本服務需透過 Meta Threads 帳號進行 OAuth 2.0 授權登入。您授予 Rethreads 以下存取權限：

  • 讀取您 Threads 帳號的貼文內容與媒體
  • 讀取您 Threads 帳號的回覆內容
  • （選用）透過 Google OAuth 發布內容至您的 Blogger

您可以隨時在 Threads 設定中撤銷本服務的授權。撤銷後，我們將停止同步您的資料，您也可以申請刪除已儲存的所有資料。

您須對自己帳號的所有使用行為負責，並確保帳號安全。若發現帳號遭到未授權使用，請立即通知我們。

03

使用規範

使用本服務時，您同意不進行以下行為：

  • 嘗試繞過、破解或干擾本服務的技術限制或安全措施
  • 使用本服務存取他人帳號資料或未授權的資料
  • 利用本服務發布違法、誹謗、侵權或有害內容
  • 以自動化方式大量呼叫本服務 API，超過合理使用範圍
  • 將本服務用於任何違反 Meta 平台條款或 Google 服務條款的行為
  • 複製、反編譯或對本服務進行逆向工程

違反上述規範可能導致帳號立即終止，且不予退款。

05

內容所有權

您在 Threads 上的貼文與回覆，其著作權歸屬依 Meta 服務條款規定。Rethreads 不主張對您內容的任何所有權。

您透過本服務生成的匯出檔案（PDF、Markdown 等）歸您所有，您可自由使用、分享或發布。

您授予 Rethreads 有限、非獨占的授權，僅用於提供您所要求的服務功能（如暫存、轉換格式、發布至 Blogger）。此授權不包含任何商業使用或對外分享的權利。

本服務依「現狀」提供，我們不保證服務不中斷或完全無誤。以下情況我們不承擔責任：

  • 因 Meta 或 Google API 政策變動導致部分功能受限或無法使用

  • 使用者因使用匯出內容所產生的任何法律責任

我們的最大賠償責任不超過您在發生問題前 3 個月內實際支付的服務費用。

07

服務中止與終止

您可以隨時申請刪除帳號，我們將在 7 個工作日內清除您的所有資料。

若您違反本條款，我們有權立即暫停或終止您的帳號，且不另行通知。

若我們決定終止本服務，將提前 30 天發出通知，並提供資料匯出功能，讓您備份自己的資料。付費用戶將依剩餘天數比例退款。

  • Meta 服務條款及 Threads 使用條款
  • Google 服務條款及 Blogger 政策（如使用 Blogger 發布功能）

Rethreads 與 Meta 及 Google 無從屬關係，我們無法控制這些平台的政策變動。若上述平台修改 API 政策導致本服務功能受限，我們將盡力提供替代方案，但不承擔相關損失責任。
