01

服務簡介

Rethreads（以下簡稱「本服務」、「我們」）是一個內容管理工具，協助使用者透過 Meta Threads 官方 API 授權，將自己帳號的貼文與回覆匯出、整理，並發布至 Blogger 或以 PDF、Markdown 格式下載保存。

本服務僅存取使用者主動授權的資料，不會在未經授權的情況下存取任何帳號資訊。

重要聲明： Rethreads 不收集、不儲存、不販售您的個人識別資訊給任何第三方。我們取得的所有資料，僅用於提供您所要求的服務功能。
02

收集的資料

我們僅收集提供服務所必要的最少量資料：

資料類型 來源 用途
Threads 帳號 ID Threads OAuth 授權 識別使用者帳號，同步對應資料
貼文內容與媒體 Threads API（僅您自己的帳號） 提供匯出、整理、發布功能
回覆內容 Threads API（僅您自己的帳號） 提供回覆整理與 AI 分析功能
Blogger 帳號授權 Token Google OAuth 授權（選用） 代表您將內容發布至 Blogger
服務使用紀錄 系統自動產生 改善服務品質、除錯

我們不會收集您的 Threads 密碼、信用卡資訊、或任何與服務功能無關的個人資料。

03

資料使用目的

您的資料僅用於以下目的：

  • 執行您所要求的匯出、備份、發布等功能
  • 提供 AI 輔助分析與內容擴寫功能（僅在您主動啟用時）
  • 同步並快取您的 Threads 貼文以提升服務效能
  • 發送服務相關通知（如匯出完成、同步完成）
  • 偵測並防範異常使用行為，保護帳號安全
  • 改善服務功能與使用者體驗

我們不會將您的內容用於訓練 AI 模型，亦不會用於任何廣告目的。

04

第三方 API 整合

本服務整合以下第三方 API，各項整合均遵守其對應的服務條款與隱私政策：

服務 用途 資料範圍
Meta Threads API 讀取您的貼文與回覆 僅限您授權帳號的公開與私人內容
Google Blogger API 發布文章至您的 Blog（選用） 僅在您連結 Blogger 後啟用
Anthropic Claude API AI 主題分類、內容擴寫（選用） 您選擇送出分析的貼文內容
關於 Meta Platform 政策： 本服務遵守 Meta 平台條款（Meta Platform Terms）及開發者政策。我們僅請求服務功能所必要的最低權限範圍，並不會將透過 Threads API 取得的資料用於任何超出本政策所述目的之行為。
05

資料儲存與安全

我們採取以下措施保護您的資料安全：

  • 所有傳輸資料均透過 HTTPS/TLS 加密
  • OAuth Access Token 以加密方式儲存，不以明文保存
  • 快取的貼文資料僅保留至您刪除帳號或主動清除為止
  • 生成的匯出檔案（PDF、Markdown）在下載後 24 小時內自動從伺服器刪除
  • 定期進行安全稽核與漏洞掃描

您可以隨時在帳號設定中申請刪除所有資料，我們將在 7 個工作日內完成清除。

06

資料分享政策

我們不會販售、出租或以任何商業目的分享您的個人資料。以下情況除外：

  • 您明確要求將內容發布至第三方平台（如 Blogger）
  • 法律要求或政府機關依法定程序請求
  • 為保護本服務、其他使用者或公眾安全所必要的情況
07

您的權利

根據適用的隱私法規，您擁有以下權利：

  • 查閱權：請求查看我們持有的您的個人資料
  • 更正權：要求更正不正確的個人資料
  • 刪除權：要求刪除您的所有資料（含已快取的貼文）
  • 撤銷授權：隨時可在 Threads 設定中撤銷本服務的 API 存取授權
  • 資料可攜權：要求以結構化格式匯出您在本服務的資料

如需行使上述權利，請透過本政策第 11 條的聯絡方式與我們聯繫。

08

類型用途保留期限

09

未成年人保護

本服務不針對 13 歲以下兒童設計，亦不故意收集其個人資料。若您發現本服務可能無意間收集了未成年人的資料，請立即聯絡我們，我們將盡速刪除相關資料。

10

政策變更

我們可能因應法規變動或服務更新而修訂本隱私政策。重大變更時，我們將以以下方式通知您：

  • 在本頁面頂部更新「最後更新日期」
  • 透過電子郵件或服務內通知告知重大變更
  • 在服務內顯示明顯的變更提示橫幅

繼續使用本服務即表示您同意更新後的隱私政策。若不同意，請停止使用本服務並申請刪除帳號資料。
